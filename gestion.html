JAVA

Estado:

Activo

Es un bróker de mensajería que ofrece caracteristicas de clustering con la facilidad de conectarse a numerosos protocolos y plataformas. Admite una persistencia muy rapida con JDBC de alto rendimiento. Diseñado para clustering de alto rendimiento, comunicación cliente servidor.

JAVA

Estado:

Activo

Es un servicio que permite una coordinación distribuida altamente confiable. Permite el manejo de informacion de configuración, proporcionar sincronización distribuida y proporcionar servicios grupales.
