HERRAMIENTAS - BASES DE DATOS - APACHE

Apache Cassandra


Tipo:

Base de datos, distribuida

Lenguaje:

JAVA

Estado:

Activo

Version:

3.11.1 (2017-10-08)

Web:

cassandra.apache.org

Descripcion:

Es una base de datos NoSQL distribuida y basada en un modelo de almacenamiento la cual permite grandes volúmenes de datos en forma distribuida. La arquitectura distribuida de Cassandra está basada en una serie de nodos iguales que se comunican con un protocolo P2P con lo que la redundancia es máxima.

Apache CouchBD

Tipo:

Bases de datos

Lenguaje:

Erlang

Estado:

Activo

Version:

2.1.1 (2017-11-07)

Web:

couchdb.apache.org

Descripcion:

Gestor de bases de datos NoSQL que emplea JSON para almacenar datos, JavaScript como lenguaje de consulta por medio de MapReduce bajo clústeres de servidores distribuidos.Un clúster CouchBD mejora la configuración de nodos con mayor capacidad y alta disponibilidad, mediante diferentes servidores o maquinas virtuales.
