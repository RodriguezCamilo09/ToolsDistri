Apache BookKeeper™


Tipo:

Servicio de registro replicado confiable

Lenguaje:

JAVA

Estado:

Activo

Version:

4.2.3 (2014-07-27)

Web:

bookkeeper.apache.org

Es un servicio de registro replicado confiable. Se puede usar para convertir cualquier servicio independiente en un servicio replicado de alta disponibilidad. BookKeeper está altamente disponible (no hay un solo punto de falla), y se escala horizontalmente a medida que se agregan más nodos de almacenamiento.

Apache Falcon™

Tipo:

Plataforma de gestión y procesamiento de datos

Lenguaje:

JAVA

Estado:

Activo

Version:

0,7-incubating (2014-12-05)

Web:

falcon.apache.org

Es una solución de gestión y procesamiento de datos para Hadoop diseñada para movimiento de datos, coordinación de canalizaciones de datos, gestión del ciclo de vida y descubrimiento de datos. Falcon permite a los consumidores finales incorporar rápidamente sus datos y sus tareas de administración y procesamiento asociadas en los clústeres de Hadoop.
